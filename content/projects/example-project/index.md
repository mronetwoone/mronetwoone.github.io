# 示例项目：内容驱动卡片

> 这是明确标记的示例内容，仅用于验证网站框架，不代表真实项目经历。

## 验证目标

这个页面用于检查从 `data/projects.json` 到首页卡片，再到统一 Markdown 阅读页的完整流程。

![示例项目结构图](images/cover.svg)

### 添加内容的最小步骤

1. 在 `content/projects/` 中新建项目文件夹。
2. 放入 `index.md` 和相关图片。
3. 在 `data/projects.json` 中增加一条记录。

| 检查项 | 预期结果 |
| --- | --- |
| JSON 数据 | 首页自动出现卡片 |
| Markdown 文件 | 点击卡片后正常渲染 |
| 相对图片 | 从内容目录正确加载 |

```html
<p>代码块会被安全地显示为文本。</p>
```

可以通过[返回首页](../../../index.html)继续查看其他示例内容。
